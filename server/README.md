# Personal Server

Personal server for data portability. 

## Deploy to Replicate

### Prerequisites
- Replicate account and API token
- Python 3.8+ installed locally

### Steps

1. **Install Replicate CLI**
   ```bash
   pip install cog
   ```

2. **Login to Replicate**
   ```bash
   cog login
   ```

4. **Deploy the model**
   ```bash
   cog push r8.im/vana-com/personal-server
   ```
